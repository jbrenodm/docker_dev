
## PHP 7.3 APACHE ##
    * Criar IMAGEM 
        $ docker build -t jbrenodm/php-apache:7.3 -f $PWD/dockerfiles/php7.3-apache.dockerfile .
    * Enviar para o DockerHub
        $ docker login 
        $ docker push jbrenodm/php-apache:7.3 
    * Criar CONTAINER
        $ docker run -d --name php7.3-apache8787 -p 8787:80 -v $PWD/projetos/php7:/var/www/html jbrenodm/php-apache:7.3
